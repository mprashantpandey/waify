5) High-Value Ongoing Checks (Release Gate)

Before every release

  • Run migration dry-check + rollback check in staging.
  • Run focused webhook, template send, billing, and inbox regression tests.
  • Confirm no duplicate toast/flash regressions on key pages.
  • Validate mobile layout on onboarding, settings, team, billing, inbox.

Weekly ops review

  • Review alert center unresolved critical/warning trends.
  • Audit failed webhook/job replay success ratio.
  • Inspect top Meta API errors by template/campaign.
  • Verify pricing snapshot/import recency and invoice correctness samples.
